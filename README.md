# Hostel Management System

This is a web-based Hostel Management System developed as an academic project.

## Description
The system helps manage hostel operations such as student registration, room allocation, and fee tracking.

## Features
- Student registration
- Hostel room management
- Admin dashboard
- Fee tracking system

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
