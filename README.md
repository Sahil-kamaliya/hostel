# Hostel Management System (PHP + MySQL)

This project is a Hostel Management System developed using PHP and MySQL. It provides functionality for both Admin and Students to manage hostel-related activities such as registration, queries, student records, and facilities.

The main purpose of this project is to understand backend development, database integration, and CRUD operations in a practical way.

## Tech Stack

- PHP  
- MySQL  
- HTML5  
- CSS3  
- JavaScript (Basic)  
- XAMPP (Local Server)  

## Features

Admin Panel

- Admin login authentication  
- Add new students  
- Edit student details  
- Delete student records  
- View all students  
- Manage student queries  
- Dashboard overview  

User Panel (Students)

- User registration and login  
- View hostel details  
- Submit queries or issues  
- Edit or delete queries  
- View facilities and gallery  
- Profile management  

General Features

- Image gallery  
- Hostel facilities section  
- Basic responsive UI  
- Database integration

## Getting Started

1. Clone the repository:
git clone https://github.com/Sahil-kamaliya/hostel.git

2. Move the project to server directory (htdocs in XAMPP)

3. Start Apache and MySQL from XAMPP

4. Open phpMyAdmin and create a database (for example: hostel)

5. Import the database file:
hostel.sql

6. Run the project in browser:
http://localhost/hostel/user/index.php

## Admin Panel

http://localhost/hostel/admin/admin_login.php
